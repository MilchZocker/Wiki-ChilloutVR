Magica Cloth V1 in CVR

In this blog post, I will explain how to use Magica Cloth on avatars in ChilloutVR. Magica Cloth is a Unity asset that allows you to create realistic cloth simulations for your characters, such as skirts, hair, capes, etc.

Dependencies

You will need Unity 2021.3.23f1, latest CCK, Burst, Collections, Mathematics and the Magica Cloth V1 packages to follow this tutorial.

 

Pre Setup

First, you need to import your avatar model into Unity and set up the basic components such as Animator, Avatar Descriptor and Avatar Settings. Then, you need to import the Magica Cloth package from the Asset Store and make sure you have the Magica Cloth Support package from the CCK Module Workshop installed.

 

Setup

Next, you need to add the Magica Cloth component to your avatar. This component will let you define the cloth settings such as gravity, wind, stiffness, damping, etc. You can also adjust the quality and performance of the simulation by changing the update rate and solver iteration parameters.


Magica Mesh Cloth

When using Magica Mesh Cloth, you need to add the Magica Virtual Deformer and Magica Render Deformer component to your avatar's mesh object. This component will create a virtual mesh that will be used for the cloth simulation.


Now you can add the Magica Mesh Cloth component, you can select from a list of preset to get a general configuration that you can tweak further more.

 

Next, you can add Magica Collider component's to your avatar. This component will let you define colliders that will interact with the cloth simulation.


Magica Bone Cloth
 

Next, you can add Magica Collider component's to your avatar. This component will let you define colliders that will interact with the cloth simulation.

 

You can add the Magica Bone Cloth component to your avatar's responding bones. This component will let you define the cloth parts that will be simulated by Magica Bone Cloth. Now you can select the Bones that should get influenced by Bone Cloth, and use the Auto Select button under Culling to Pause Simulation when not rendered. Then you can select from a list of preset to get a general configuration that you can tweak further more.

Next, you can add Magica Collider component's to your avatar. This component will let you define colliders that will interact with the cloth simulation.

 

Finishing

That's it! You have successfully added Magica Cloth simulation to your avatar in chilloutvr. You can test your avatar in-game by uploading it using CCK 3.5 and selecting it in the Avatars Tab. You should see your cloth parts moving realistically with physics.
