ChilloutVR Interactable

What are interactables and how do they work in ChilloutVR?

Interactables are objects that can be manipulated by the player using their VR controllers or mouse and keyboard. They can have different behaviors and effects depending on how they are configured by the world creator. For example, an interactable door can be opened and closed by grabbing and pulling it, an interactable button can activate a sound or a light when pressed, an interactable lever can change the state of another object when flipped, etc.

 

Interactables are created using the CCK, which is a set of tools and scripts that allow world creators to customize their worlds using Unity. The CCK provides a variety of components that can be attached to any game object to make it an interactable. These components define the properties and functions of the interactable, such as its appearance, physics, sound, animation, interaction mode, etc.

 

How do interactables work in ChilloutVR? When a player approaches an interactable object, they will see a highlight around it and a tooltip. The player can then grab or touch the object using their VR controller or mouse cursor, depending on the interaction mode of the object. The object will respond accordingly to the player's input, such as moving, rotating, scaling, changing color, playing sound, etc. The object will also send events to other scripts or objects in the scene when certain conditions are met, such as when it is grabbed, released, pressed, etc. These events can be used to trigger actions or effects in the world, such as opening a door, turning on a light, playing a video, etc.
