Facetracking capabilities

 

Tobii [Official]

Tobii Eyetracking can be enabled in settings>Implementations>Enable Eye tracking

 

SRanipal (Vive) [Official]

Vive Facetracking with SRanipal can be enabled in settings>Implementations>Enable Vive Facetracking, SRanipal has its own standard of Visemes/Blend shapes to use. 

 

VRCFaceTracking [Unofficial]

VRCFT is an Application that communicates via OSC with CVR
Any kind of Eye/Facetracking supported by VRCFacetracking can be used in ChilloutVR.

Requirements:
VRCFacetracking version 5.1.1.0!

VRCFaceTracking 5.1.1.0 compatible module/s for the used Hardware/Software

Kafe's OSC Mod

VRCFacetracking compatible Avatar


VRCFT2CVR (VRCFT Unity) [Unofficial]

Unity native VRCFT implemented as a MelonLoader Mod
Any kind of Eye/Facetracking supported by VRCFacetracking can be used in ChilloutVR as long as the module/s get recompiled for VRCFT Unity.

Requirements:
VRCFT2CVR

Recompiled Modules for the used Hardware/Software, Make Sure you also get the necessary Libs for each module!

VRCFacetracking compatible Avatar


ProjectBabble / EyeTrackVR (Direct) [Unofficial]

Project Babble is an opensource mouth tracking project designed to work with any existing VR headset.
EyeTrackVR is an Open Source, DIY and Affordable, Virtual Reality Eye Tracking Platform.

Both can also be used via VRCFaceTracking Modules as described above.

Requirements:
ProjectBabble / EyeTrackVR

In settings for Babble / ETVR, set the OSC address port from its default 8888 to 9000. Make sure to set the location prefix to /avatar/parameters.

Kafe's OSC Mod

VRCFacetracking compatible Avatar


Pico  4 Pro/Enterprise (Streamer/Business Streamer)

Pico 4 Pro/Enterprise Eye/Mouth tracking via offical Streamer 


Requirements:
Enable both Eye Tracking and Lip Tracking on Pico Settings, those can be found under LAB tab

VRCFT2CVR

VRCFT2CVRPicoModule

VRCFacetracking compatible Avatar
 

(Pico 4 Enterprise requires renaming ""…\BusinessStreaming\BusinessStreamingUW\driver\bin\win64\pico_et_ft_bt_bridge.exe" to “.pico_et_ft_bt_bridge.exe” to not interfere with module.)

(Pico 4 Pro requires change its facetracking to legacy protocol, find the file at “C:\Users\{YourUsername}\AppData\Roaming\PICO Connect\setting.json” and change faceTrackingTransferProtocol to 2)


HP Reverb G2 Omnicept Edition (WMR Omnicept Runtime)

Integrated sensors measure muscle movement, gaze, pupil size and pulse and seamlessly transfers data to the HP Omnicept 


(For Eyetracking you can also use CVR's (limited) build-in Tobii support)

Requirements:

Omnicept Runtime (Remember to run Eye Tracking Calibration after installation)

Omnicept Tray (Disable Ask for Permission to use Sensor Data)

Glia LibViewer (Application will display the Raw Facecamera image)

OBS (Capture the Glia LibViewer Application then Start the Virtual Camera)

ProjectBabble

VRCFT2CVROmniceptModule (Omnicept Module is Only For Eyetracking! using this module you will get seperate Eye Blink, Gaze, PupilSize)

VRCFT2CVRProjectBabbleModule (to recive the result from ProjectBabble)

HRtoVRC (to send Heart Rate via OSC to Kafe's OSC Mod)

VRCFacetracking compatible Avatar
