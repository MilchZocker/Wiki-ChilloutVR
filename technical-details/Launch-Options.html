Launch/Startup Options

The following Launch Options can be added in the properties tab on steam.

 

Compatibility

Disables AVPro/Videoplayers entirely:

--disable-videoplayers

Disables AVPro/Videoplayer Hardware Decoding:

--disable-vphd

Uses VLC instead of AvPro (recommended for Linux Users):

--enable-libvlc

Starts the Game without Steam (breaks DLC's):

-skipsteam

Force VR Launch:

-vr

Optional Features

Use OpenXR instead of OpenVR:

--force-openxr

Allows to define an affinity mask:

--affinity=<mask>

Allows to define a path used for Content Cache:

--content-cache-path="F:\example\directory\"
